1️⃣ What is the difference between var, let, and const?

var, let, and const are used to declare variables in JavaScript.
var is the old way and it can be redeclared and updated.
let is a modern version that can be updated but cannot be redeclared in the same scope.
const is used for values that should not change, so it cannot be updated or redeclared after it is declared.

2️⃣ What is the spread operator (...)?

The spread operator (...) is used to expand or copy values from an array or object.
It is commonly used to combine arrays or clone objects in a cleaner way.

Example: It can merge two arrays into one.

3️⃣ What is the difference between map(), filter(), and forEach()?

All three are array methods used to loop through arrays.

map() creates a new array by transforming each element.
filter() creates a new array containing only the elements that match a condition.
forEach() simply runs a function for each element but does not return a new array.

4️⃣ What is an arrow function?

An arrow function is a shorter and cleaner way to write functions in JavaScript.
It uses the => syntax and is commonly used in modern JavaScript code.

Example: const add = (a, b) => a + b;

5️⃣ What are template literals?

Template literals are a way to write strings using backticks (`).
They allow us to easily insert variables inside a string using ${}.

Example: `My name is ${name}`
